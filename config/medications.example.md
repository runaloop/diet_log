# Таблетки и БАДы

> Локальный файл. Скопируй в `config/medications.md` (`setup.sh` делает это автоматически)
> и заполни своими препаратами. `config/medications.md` в `.gitignore` — не коммитится,
> не синкается, остаётся только на этой машине.

Форматы периода:
- `бессрочно` — принимать каждый день без ограничения по дате
- `2026-04-01..2026-06-30` — курс с конкретными датами начала и конца
- `пн,ср,пт` — только в указанные дни недели (пн/вт/ср/чт/пт/сб/вс)
- `2026-04-01..бессрочно` — начиная с даты, без конца

Примечание `через день` — приём через день от даты начала периода.

| Препарат | Доза | Приём | Период | Примечание |
|----------|------|-------|--------|------------|
| Витамин D3 | 2000 МЕ | утро | бессрочно | |
| Омега-3 | 1000 мг | утро | 2026-01-01..бессрочно | через день |
| Магний | 400 мг | вечер | пн,ср,пт | |

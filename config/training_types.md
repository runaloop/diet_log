# Типы тренировок и компенсация углеводов

Компенсация показывает сколько калорий углеводов нужно добавить к базовому расходу для восстановления гликогена.

| Тип                    | Код      | Компенсация углеводов |
| ---------------------- | -------- | --------------------- |
| Восстановительная / Z1 | Z1       | 15%                   |
| Aerobic base / Z2      | Z2       | 30%                   |
| Tempo / Z3             | Z3       | 50%                   |
| Threshold / Z4         | Z4       | 70%                   |
| VO2max / sprint / Z5   | Z5       | 90%                   |
| Силовая тренировка     | strength | 55%                   |
| HIIT / интервалы       | hiit     | 70%                   |
| Прогулка               | walk     | 10%                   |

## Пример расчёта

Тренировка Z2 на 700 ккал:
- Компенсация: 700 × 30% = 210 ккал
- В граммах углеводов: 210 ÷ 4 = 52.5г

## Примечание

Если тип тренировки не указан явно — компенсация не применяется.

## Дорожка 4км/ч

Если пользователь пишет `дорожка 4км/ч` без указания калорий, считать калории по скорости, текущему весу из `user.md` и длительности.

Использовать формулу ходьбы через MET:
- для `4 км/ч` брать `3.0 MET`;
- `ккал = MET * 3.5 * вес_кг / 200 * минуты`.

Пример для веса `85.9 кг`:
- `дорожка 4км/ч 1ч` → `3.0 * 3.5 * 85.9 / 200 * 60 = 270.6 ккал`
- `дорожка 4км/ч 2ч` → `541.2 ккал`

Для компенсации углеводов считать такую запись как `Прогулка` с компенсацией `10%`.
